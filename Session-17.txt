1.Κάνοντας μία GET και να χρηστιμοποιήσουμε μόνο τα Endpoints.

2.Το response το type που θα έχει είναι image/(png, img, jpeg) ανάλογα με την κατάληξη που έχει η εικόνα που κάνουμε GET.

3.Κάνοντας enable το caching και έχωντας στο head response το max-age=86400.Με την Request.cache αν υπάρχει αντιστοίχιση με την cache και είναι fresh θα επιστραφεί από την cache, αν υπάρχει παλιό request στην cache θα πάει να κάνει αντιστοίχιση και άμα δεν έχει αλλάξει θα το γυρίσει από την cache, αλλιώς θα πάρει το καινούργιο request και θα το περάσει στην cahce.

4.Είναι εφικτό να έχουμε μία εφαρμογή που κάνει μόνο GET requests, αλλά δεν είναι επιθυμητό γιατί δεν περιγράφονται σωστά οι πράξεις που πάμε και κάνουμε στην εφαρμογή. 

5.Είναι εφικτό να έχουμε μία εφαρμογή που κάνει μόνο POST requests, όμως δεν είναι καλό γιατί τα δεδομένα που θα μας επιστρέφει δεν ξέρουμε αν θα είναι εύκολα αναγνώσιμα και έτσι η ικανότητα για να ανακαλύψουμε άλλα πράγματα μπορεί να είναι περιορισμένη.

6.412 Precondition Failed ή 422 Unprocessable Entity (WebDAV) αν το syntax είναι σωστό.

7.502 Bad Gateway

8.Μπορεί να είναι GET request, αλλά δεν πρέπει και δεν είναι ασφαλές, γιατί με την GET request τα δεδομένα μας είναι εκτεθειμένα.

9.H PUT request είναι indempotent.Δηλαδή άμα καλέσουμε την PUT πολλές φορές θα γυρίσει πάντα το ίδιο αποτέλεσμα, ενώ η POST request άμα την καλέσουμε πολλές φορές θα δημιουργήσει πολλές φορές το ίδιο αποτέλεσμα.

10.Θα χρησιμοποιούσαμε την PUT request όπου στο body της request θα βάζαμε τα στοιχεία του αρχέιου.
